# Deadline Guardian

Deadline Guardian is an immersive, high-performance productivity workspace designed to defend target milestones, manage complex schedules, and protect critical timeline buffers. It features deep work diagnostics, automated task slicing, chronological daily plan mapping, and a zen-inspired immersive focus environment.

## Key Features

- **Sentinel Diagnostics**: Actively track individual work pressures, calculate dates remaining, count subtask completions, and evaluate overall milestone risk factors.
- **Slicing Checklists**: Intelligently analyze complex, multi-day goals and break them down into granular, actionable subtasks.
- **Optimal Prioritizer**: Evaluate and resolve resource overlaps, portfolio-wide deadline collisions, and timeline conflicts.
- **Daily Buffer Blocks**: Construct smooth chronological schedules that respect bedtime boundaries, buffer zones, and active focus thresholds.
- **Immersive Focus Room**: A dedicated, distraction-free environment equipped with customizable ambient background audio controls, date/time tracking, a click-to-toggle 12h/24h (military) clock format, and a true immersive fullscreen mode.

## Tech Stack & Architecture

- **Frontend**: React (TypeScript), Vite, Tailwind CSS, Lucide Icons, and Motion animations.
- **Backend**: Express server configuration serving assets and handling backend telemetry processing.
- **State Management**: Robust client-side persistent hooks ensuring active commitments, checklist progresses, and daily schedule plans remain intact.

## Getting Started

### Development

To start the local development server:

```bash
npm run dev
```

The application will be accessible in your web browser.

### Building

To compile the application into static assets and bundle the backend entry point:

```bash
npm run build
```

This generates compiled, production-ready assets under the `dist/` folder.
